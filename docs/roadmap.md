# Project Roadmap

This roadmap outlines the planned development of the Library Management System.

## Sprint 1 (Completed)
- Project setup  
- Database configuration  
- Book entity  
- CRUD operations  
- Basic tests  
- Documentation  

## Sprint 2
- User management (librarians and borrowers)  
- User entity and repository  
- Basic user CRUD  

## Sprint 3
- Borrowing and returning books  
- Loan entity  
- Loan workflow logic  

## Sprint 4
- Search and filtering  
- Query parameters for books  
- Category and metadata improvements  

## Sprint 5
- Authentication and roles  
- Login system  
- Role-based access control  

## Sprint 6
- Dashboard and improvements  
- Statistics  
- UI refinements  
- Final documentation
